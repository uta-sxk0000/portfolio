# Sagar Khadka Portfolio

Modern personal portfolio website for **Sagar Khadka**, a Computer Science student and aspiring Software Engineer focused on software development, AI, cybersecurity, systems, and full-stack projects.

Live site: [khadkasagar.name.np](https://khadkasagar.name.np)

## About

This portfolio is designed for internship applications, recruiter review, academic projects, and professional networking. It presents my technical background, project work, resume, and contact links in a clean premium dark-tech interface.

## Features

- Responsive portfolio website built with HTML, CSS, and JavaScript
- Premium dark UI with glassmorphism, glow effects, and animated background particles
- Main profile card with professional headshot
- Project section with category filtering and custom project visuals
- Resume preview and direct resume download
- Contact section with Email, GitHub, LinkedIn, and Instagram links
- Command Center feature for quick navigation with `Ctrl + K`
- Smooth scrolling, active navbar state, scroll progress indicator, and mobile hamburger menu
- Cloudflare Pages-ready static site with no backend or build step required

## Tech Stack

- HTML5
- CSS3
- JavaScript
- SVG project illustrations
- Google Fonts
- Static deployment through Cloudflare Pages

## How To Run Locally

Because this is a static website, no installation is required.

1. Clone the repository.
2. Open `index.html` in a browser.
3. Edit `index.html`, `style.css`, and `script.js` as needed.

## Deployment

This project works directly on Cloudflare Pages.

Recommended Cloudflare Pages settings:

- Build command: none
- Output directory: root directory
- Root file: `index.html`

## Customization Notes

- Replace `resume.pdf` with the latest resume while keeping the same filename.
- Update project GitHub/live links in `index.html` when repositories or demos change.
- Replace `assets/profile/sagar-profile.png` to update the profile photo.
- Edit colors, spacing, and animations in `style.css`.
